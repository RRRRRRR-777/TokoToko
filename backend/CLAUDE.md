# CLAUDE.md

このファイルは、backend ディレクトリで作業する際の Claude Code へのガイダンスを提供します。

## 🔴 最重要ルール - ドキュメント参照順序

backend ディレクトリで作業する場合、以下の順序でドキュメントを参照してください：

1. **AGENT.md**（backend/AGENT.md）
   - backend 作業の行動指針
   - コミットルール
   - 設計要約
   - **まずこのファイルを読むこと**

2. **backend/docs/***
   - 詳細設計ドキュメント
   - 必要に応じて参照

## ルートの CLAUDE.md との関係

- **ルートの CLAUDE.md**: iOSアプリ（TekuToko）全体の開発ルール
- **backend/CLAUDE.md（このファイル）**: backend 専用のルール

backend で作業する場合は、**このファイル（backend/CLAUDE.md）のルールを優先**してください。
